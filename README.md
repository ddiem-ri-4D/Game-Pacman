# 1. Install Python's required packages

- **Python** _(3.8.5)_
- **PySide2** _(5.15.2)_
- **pygame** _(2.0.1)_
- **numpy** _(1.19.4)_

# 2. Run

- Inside the **pacman** folder, open terminal and run command:
  `python3 main.py`

  - If your operating system is **Window**, you can try with the commands below:
    > `python main.py` > `py main.py`
